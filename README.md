# Aidan's Website(s)

My website, made using Rust. :)

## Building & Running

### Perseus CLI

To build the website, you need to have the Perseus CLI installed. You can install it by running:

```sh
cargo install perseus-cli
```

To build the website, run:

```sh
perseus build
```

To run the website, run:

```sh
perseus serve
```
