# Aidan's Website(s)

My website, made using Kotlin ([Ktor](https://ktor.io/) + [kotlinx.html](https://github.com/Kotlin/kotlinx.html)) and
Tailwind. :)

## Building & Running

To build the website, run:

```sh
gradle build
```

To run the website, run:

```sh
gradle run
```
